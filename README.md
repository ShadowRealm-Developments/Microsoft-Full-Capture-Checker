# 📦 How to Install

1. Install Python 3.10 or higher

2. Download or clone the repository

```bash
git clone <repo_url>
```

3. Open the project folder

```bash
cd <project_folder>
```

4. Install dependencies

```bash
pip install -r requirements.txt
```

5. Run the application

```bash
python main.py
```
